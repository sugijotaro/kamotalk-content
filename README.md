# kamotalk-content

This repository hosts the content for KamoTalk.
It provides an API of articles and hosts the markdown content.

## Accessing Content

- **Articles JSON**: `https://sugijotaro.github.io/kamotalk-content/api/articles.json`
- **Markdown Files**: `https://sugijotaro.github.io/kamotalk-content/content/articles/{filename}.md`

## Development

Run `npm run build:articles` to generate the articles JSON locally.
